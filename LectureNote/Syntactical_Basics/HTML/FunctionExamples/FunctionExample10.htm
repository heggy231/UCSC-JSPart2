Here, the function addSomeNumber() is defi ned twice. The fi rst version of the function adds 100
    to the argument, and the second adds 200.

    When the last line is called, it returns 300 because the
    second function has overwritten the first.

    As mentioned previously, it is  possible to simulate overloading of methods by checking the type and
    number of arguments that have been passed into a function and then reacting accordingly.

            alert(num + 100);
        }

            alert(num + 200) ;
